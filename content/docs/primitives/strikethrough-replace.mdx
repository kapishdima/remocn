---
title: Strikethrough Replace
description: Draw a strike line across old text, then reveal new text in its place.
---

A typography primitive that animates a strikethrough line growing left-to-right across the `from` text, then cross-fades to the `to` text. Useful for "old way / new way" comparisons.

<ComponentPreview name="strikethrough-replace" />

## Installation

<InstallBlock name="strikethrough-replace" />

## Usage

```tsx
import { StrikethroughReplace } from "@/components/remocn/strikethrough-replace";

<StrikethroughReplace
  from="Slow setup"
  to="Instant setup"
  lineColor="#ff5e3a"
  fontSize={72}
/>
```

## Notes

<Note title="Three phases">
  The animation runs in three phases tied to `durationInFrames`: draw the strike line (0–40%), cross-fade to the new text (40–60%), then hold (60–100%).
</Note>
