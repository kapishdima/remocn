---
title: Simulated Cursor
description: Animated mouse cursor that moves between waypoints with optional click feedback.
---

A UI primitive that renders a macOS-style cursor (inline SVG, no external assets) and moves it between a list of points. Each waypoint can hold for a configurable number of frames and optionally trigger a click animation: a brief scale dip plus an expanding SVG ripple.

## Usage

```tsx
import { SimulatedCursor } from "@/components/remocn/simulated-cursor";

<SimulatedCursor
  points={[
    { x: 200, y: 150, hold: 20 },
    { x: 800, y: 360, hold: 25, click: true },
    { x: 1050, y: 560, hold: 20 },
  ]}
  color="#ffffff"
/>
```

## Notes

<Note title="Inline SVG, no assets">
  The cursor is drawn as inline SVG, so you don't need to copy anything into `public/`. Drop the component into any composition and it just works.
</Note>

<Note title="Coordinate space">
  `x` and `y` are in pixels relative to the composition size (e.g. 1280x720). Coordinates larger than the canvas will still render but be clipped by `overflow: hidden`.
</Note>
