**remocn** is an open registry of animations, transitions, backgrounds, and complete scene compositions for [Remotion](https://www.remotion.dev). It follows the shadcn philosophy: components are copied directly into your project, so you fully own and customize them.

## What's inside

- **Primitives** — single-purpose animations like `BlurReveal`, `Typewriter`, `MeshGradientBg`.
- **Compositions** — full scenes built from primitives.
- **Live previews** — every component page mounts a real Remotion `<Player>` you can scrub and customize.

## Install a component

```bash
npx shadcn@latest add remocn/blur-reveal
```

The file is copied into your project. No runtime dependency on remocn — just Remotion and React.
