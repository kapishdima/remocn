---
title: Blur Reveal
description: Text fades in from a heavy blur into sharp focus.
---

A typography primitive that animates `opacity` from 0 to 1 and `filter: blur()` from a configurable amount down to 0. Useful for title cards and intro shots.

<ComponentPreview name="blur-reveal" />

## Installation

<InstallBlock name="blur-reveal" />

## Usage

```tsx
import { BlurReveal } from "@/components/remocn/blur-reveal";

<BlurReveal text="Hello, world" blur={12} fontSize={72} />
```

## Notes

<Note title="Load fonts before render">
  Make sure your font is loaded before the first frame renders. Use [`@remotion/google-fonts`](https://www.remotion.dev/docs/google-fonts) or `@remotion/fonts` so the blur effect doesn't apply to a fallback system font.
</Note>

<Warning title="Performance">
  Heavy blur on large text blocks is expensive. If you stack multiple `BlurReveal`s, consider rendering them in their own `<Sequence>` to keep render time predictable.
</Warning>
