## Roadmap

- **`MeshGradientBg`** — amorphous color blobs animated with `@remotion/noise` for organic, deterministic motion. Optionally backed by GLSL shaders for high-end gradients.
- **`DynamicGrid`** — subtle moving grid (or dot pattern) used as a backdrop. Animated via `transform: translate()` for cheap rendering.
